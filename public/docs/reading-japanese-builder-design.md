Builder は**翻訳器ではなく**、Morph・Syntax・Semantic が提供する情報を統合し、各章節の japanese を

パイプライン: Data → Morph → Syntax → Semantic → **Reading Japanese(Builder)**

---

## 0. Builder の位置づけ

- Morph/Syntax/Semantic は **トークン単位の反映情報**を出す(語形・構造情報・語義)。
- **Builder は章節(verse)単位で、それらを統合し最終的な読む日本語を確定する**上位層。
- Builder は各層の**決定を代行しない**。各層が決めた結果を受け取り、**章節として組み立てる**のみ。

---

## 1. Builder の責務(最終的に決定するもの)

| 責務 | 内容 |
|---|---|
| **章節単位の japanese 確定** | chapter / verse 単位で最終的な読む日本語を決定する |
| **語の接続** | 隣接語が日本語として接続するよう統合(助詞・活用が破綻しない) |
| **章節内一貫性** | 同一章節内で同語・同構造の扱いが矛盾しないよう保証 |
| **構造反映の統合** | Morph の語形 + Syntax の構造情報 + Semantic の語義を、章節の読む日本語へ統合 |
| **L-0 品質の保証** | 読解可能・構造忠実・章節一貫・語接続・非誤解・非破損(L-0 §4)を章節単位で満たす |

- Builder は「各層の反映を**章節として成立させる**」最終責務を負う。

---

## 2. Builder が決めないもの(明確な除外)

| 除外 | 決定する層 |
|---|---|
| **Morph 判定**(語形: 彼ら/それ/者/もの・格助詞・活用) | Morph(v1) |
| **Syntax 判定**(関係詞 role/先行詞リンク/節境界) | Syntax(K-3/K-4) |
| **Semantic 判定**(文脈語義選択・idiom) | Semantic |
| **新しい意味推論** | 誰も行わない(L-0 禁止) |
| **翻訳**(自然な目標言語文の生成) | 対象外 |
| **日本語語順の自然化**(語順再構成) | 対象外(K-5 で不採用) |

- Builder は各層の**決定を受け取り統合する**のみ。決定そのものは各層に属する。

---

## 3. Builder の入力(責務として整理・利用順序ではない)

| 入力 | 内容 | Builder の利用責務 |
|---|---|---|
| **Data** | 起点の代表語(bible_data.japanese の元) | 統合の**起点**(削除・多義化しない) |
| **Morph 結果** | 読む語形(彼ら/それ/者/もの/私自身/格助詞) | 章節に組み込む語形素材 |
| **Syntax 結果** | 構造情報(関係詞 role/先行詞リンク/節境界) | 構造反映の素材(語順は変えない) |
| **Semantic 結果** | 文脈語義・idiom | 語義選択の素材 |

- Builder は各層の**内部判断に立ち入らない**。提供された結果を「章節を組む材料」として利用する。
- 利用は**順序ではなく責務**: どの層の情報も、章節の読む日本語を成立させるために統合される。

---

## 4. Builder の出力(唯一)

- **各 verse の最終的な読む日本語**(bible_data.japanese の deliverable としての確定値)。
- **対象は japanese のみ**。
- **対象外**: StudyPanel 用表示・研究補助情報(K-5 StudyPanel Integration は別責務・別出力)。
- 起点の Data 代表語は参照・比較用に保持される(H-3b)。Builder の出力は「読む日本語」の確定値であり、
  Data 代表語(起点)とは役割が異なる。

---

## 5. 評価単位(品質保証単位)

- **Builder が品質保証する単位 = verse(章節)**(L-0 §8 に従う)。
- 理由: 最終成果物が「各章節の japanese」であり、読者は章節を読む。
- token / phrase / clause は**必要条件・中間検査**の単位(局所正しさ・接続・節構造)。
  **最終保証は verse**。

---

## 6. Builder の判断原則(候補統合時の優先順位)

複数層の情報を統合する際の優先順位(上が優先):

1. **非破損(読解可能性)** — 破損形・二重助詞・意味不明を出さない(最優先)。
2. **構造忠実性** — ギリシャ語の形態・構造を隠さない。Semantic が構造を破壊しない。
3. **Morph と Syntax の無矛盾** — 語形と構造情報が矛盾しないこと。矛盾時は構造忠実を優先。
4. **章節内一貫性** — 章節内で同語・同構造が一貫。
5. **語の接続(読みやすさ)** — 最後。**自然さのために上位(構造忠実)を犠牲にしない**。

- **L-0 §7 準拠**: 「自然になった」だけでは改善としない。自然化が構造を隠すなら**悪化**。
- 原則の一語要約: **非破損 > 構造忠実 > 無矛盾 > 一貫 > 接続**。

---

## 7. Builder の対象範囲

| 対象(扱う) | 由来 |
|---|---|
| 関係詞の構造反映 | Syntax(K-3/K-4・1,658) |
| 指示詞の連体/代名詞 | Syntax(1,687) |
| 再帰代名詞の person | Semantic(ἑαυτοῦ 823) |
| 不定用法 | Semantic(τις 525) |
| 語の接続 | 章節統合 |
| 章節品質(読解可能・一貫) | 章節統合 |

| 対象外(扱わない) | 理由 |
|---|---|
| Morph の語形決定 | Morph v1 が決定(Builder は利用のみ) |
| 翻訳・自然文生成 | L-0 方針外 |
| 日本語語順再構成 | K-5 で不採用 |
| 意味推論・語義断定 | L-0 禁止 |
| StudyPanel 表示・研究補助 | 別責務(K-5) |
| 自由関係(referent 無 587)・複雑節の再構成 | Semantic / 範囲外 |

---

## 8. Builder と各 Engine の責務境界

| 層 | 決定する | 決定しない |
|---|---|---|
| **Morph** | 形態から読む語形(gender/number/case/person/mood) | 構造・語義・章節統合 |
| **Syntax** | 構造情報の提供(role/referent/節境界) | 語形・語義・章節確定・語順変更・文字列生成 |
| **Semantic** | 文脈語義の整理(選択・idiom) | 語形・構造・章節確定・意味推論 |
| **Reading Japanese Builder** | **章節としての統合・最終確定**(接続・一貫性・構造反映の統合) | 語形/構造/語義の内部判断(各層に委任)・翻訳・自然化・意味推論 |
| **Presentation** | 表示整形のみ(括弧・強調・UI) | 語・語順・確定値の変更 |

- **各層は「部分の決定」、Builder は「章節としての確定」**。責務は重ならない。

---

## 9. QA 方針

| 手段 | 内容 |
|---|---|
| **chapter / verse 単位レビュー** | 人間レビューで読解可能性・章節内一貫性・語の接続・誤解の有無を評価(L-0 §9 の人間レビュー) |
| **回帰** | Morph 回帰(morph 44,251・7 lemma)・Syntax 回帰(K-3/K-4 バイト等価)・既存 8 スイート ALL PASS |
| **差分監査** | japanese の変更が意図した章節・範囲のみであること。**重複「 2.json」混入の検出を必須**(H-5/H-6 の教訓)。実 FS・重複不在の確認 |
| **悪化判定** | L-0 §7 に従う。構造隠蔽・誤解発生・破損発生・一貫性低下・推論混入 = 悪化。**自然化単独は改善としない** |
| **Reading Japanese Policy 準拠** | L-0 の全基準(定義・反映原則・ふさわしい・翻訳化境界・改善悪化・評価単位)に照らす |

- QA は **自動(回帰・差分・破損・重複検出)+ 章節人間レビュー(理解補助・一貫性)** の二段。

---

## 10. FROZEN 条件

Builder 設計を凍結できる条件:

- §1〜§9(責務・除外・入力・出力・評価単位・判断原則・対象範囲・責務境界・QA)がすべて確定している。
- L-0(Policy)・L-1(Baseline)と整合している。
- 既存 FROZEN 資産(Morph v1・Relative Syntax K-3/K-4)を侵さない設計である。
- Builder は**統合・確定層**であり、Morph/Syntax/Semantic の**決定を代行しない**ことが明記されている。
- **翻訳・自然化・意味推論・語順再構成を含まない**。
- 出力対象が bible_data.japanese(読む日本語の確定値)のみで、StudyPanel/研究補助を含まない。
