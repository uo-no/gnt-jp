目的: bible_data.japanese の固定日本語が疑問詞・関係詞ファミリーで読者を誤解させないかを
実測し、Data / Engine / Semantic のいずれが担うべき責務かを分類する。
**本監査では修正を一切行わない**(bible_data / Engine / Presentation / lexicon 変更なし)。
全数値は NT 全巻の実測値(推測なし)。

対象データの存在は test -e で確認済み。Engine 出力は本番同一経路
(ReadingContext → ReadingEngine → PresentationPolicy)で生成。

## Phase 4: particle(は/が)付与監査

疑問詞・関係詞に Engine の particle 層が は/が を付与したケース(実測):

| lemma | 固定訳 | 付与件数 | 例 |
|---|---|---|---|
| **ὅς** | 〜する者 | **208** | 1CO 1:8 ὃς →〜する者が / 1CO 10:13 ὃς →〜する者が |
| **ὅστις** | 〜する者 | **92** | 1CO 5:1 ἥτις →〜する者が / 1TI 1:4 αἵτινες →〜する者が |
| **τίς** | 誰 | **68** | 1CO 2:11 τίς(who)→誰は / 1JN 3:2 τί(what)→誰は |
| ὅσος | 〜するだけ | 18 | ACT 10:45 ὅσοι →〜するだけが |
| πόσος | どれほど | 1 | LUK 15:17 →どれほどは |
| ὁποῖος | どのような | 1 | JAS 1:24 →どのようなは |

**疑問詞への「は」付与(τίς 68 件)** は疑問文で不自然(「誰は」)。
**関係詞への「が」付与(ὅς 208・ὅστις 92)** は関係節を主語名詞句化するもの。

## Phase 5: 関係詞特有監査(ὅς 系)

固定訳「〜する者」「〜するだけ」は **人称代名詞的な substantive(者=人)** として訳出しており、
関係節の構文性と referent の性(モノ=中性)が反映されていない。

| lemma | 固定訳 | 中性(モノ)件数 | 問題 |
|---|---|---|---|
| ὅς | 〜する者 | **482** | 中性(which/what=モノ)を「者」(人)と訳出。例: 1CO 10:16 ὃ(which)→〜する者を |
| ὅστις | 〜する者 | 5 | 同上。例: JHN 21:25 ἅτινα(which)→〜する者 |
| ὅσος | 〜するだけ | 72 | 中性(whatever things)を「〜するだけ」。例: 1CO 2:9 ὅσα →〜するだけを |

確認項目の結果:
- **人称代名詞のように見える訳か**: はい。「〜する者」は人限定で、モノ(中性 482+72)に不適合。
- **関係節として読めるか**: 部分的。「〜する者」は substantive 化しており、後続の関係節構造は
  語順フローで追う必要がある。
- **morph gender が反映されているか**: **いいえ**。固定単一訳のため gender(M/F/N)非反映。

## 責務分離の所見(修正は行わない)

判断基準(Data=安定した代表語 / Engine=形態による変化 / Semantic=文脈による選択 /
Presentation=表示のみ)に沿った分類:

| 責務 | 該当 | 内容 |
|---|---|---|
| **Data で安定(A)** | πῶς・πόσος・ποῦ | 固定訳で読者を誤解させない。現状維持で問題なし |
| **Data 代表語だが Engine 補完が望ましい(B)** | πόθεν・πότε・ποῖος・ὁποῖος | 見出しは固定訳で可。本文表示では軽微な不適合あり |
| **Engine(形態)が必要(C)** | **τίς** | gender(中性→何/なぜ、男性→誰)で意味が変わる。単一訳では 74.4% 不適合。Stage H-1 の判定 C を裏付け |
| **Engine(形態)+ Semantic(構文)が必要(C)** | **ὅς・ὅστις・ὅσος** | 関係詞。gender(モノ/人)と関係節構文の反映が必要。固定「〜する者」は人限定で中性に不適合。加えて particle 層が「が」を付与(ὅς 208件)し名詞句化 |

### particle 層との相互作用(Engine 領域)

- τίス の「誰は」(68件)は疑問代名詞への は 付与。
- ὅς/ὅστις の「〜する者が」(300件)は関係詞への が 付与。
いずれも particle 層(Engine)のルール適用範囲の問題であり、Data では解決しない。

## 結論

- **Data 単独で安全なのは A の 3 lemma(πῶς・πόσος・ποῦ)のみ**。
- **τίς(555件・unsafe 74.4%)と関係詞 3 lemma(ὅς 1408・ὅστις 139・ὅσος 111)は Data 固定訳では
  一定割合で誤り、形態(gender)・文脈(構文)による Engine/Semantic 選択が必要**。
- 目的は japanese の削除ではなく責務の明確化。C 分類の lemma は「Data が単一代表語を持ち、
  Engine が形態で変化させる」責務分離が妥当(本監査は分類のみ・修正は次段階)。
