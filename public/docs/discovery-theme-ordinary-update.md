# Phase UX-10-3D — Ordinary Theme Pool Update

**実施日**: 2026-07-14  
**変更範囲**: `assets/data/discovery-themes.json` + `search-tool.html`（データミラー同期のみ）  
**根拠**: UX-10-3C 監査（ordinary uniqueness 25% → 要改善）

---

## 目的

UX-10-3C の監査結果に基づき、ordinary シーズンの seasonal プールを  
他プールと重複しない語に差し替え、uniqueness を 25% → 100% に改善する。

## 副作用確認

旧 ordinary seasonal 語（祈り・平安・知恵）は seasonal プールからは除去されたが、  
core（祈り・平安）および discovery（知恵）に引き続き存在するため検索体験には影響しない。  
seasonal プール内での重複スキップも発生しなくなり、deduplication ロジックの負担が軽減した。
