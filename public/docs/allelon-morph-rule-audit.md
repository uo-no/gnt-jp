## 2. gender / number / case 分布

| 観点 | 分布 | 所見 |
|---|---|---|
| **gender** | masculine 99 / neuter 1 | 「互いに」は gender 中立 → gender は無関係 |
| **number** | **plural 100(全件)** | 相互代名詞は本質的に複数のみ・変化なし |
| **person** | すべて "-"(非マーク) | person 中立 |
| **case** | accusative 67 / genitive 20 / dative 13 | **主格なし**(斜格のみ・相互代名詞の性質) |
| morph | C-APM / C-GPM / C-DPM(C=相互) | 常に複数・性は M(1件のみ N) |
| english | one another 82 / each other 6 / themselves 4 / one another's 2 | 相互「互いに」 |

**要点**: ἀλλήλων は **常に複数・person 中立・gender 無関係・主格なし**。形態上の変化軸(gender/number/
person)がいずれも **不変または中立** であり、語幹を分岐させる形態的余地がない。

---

## 3. 現在の japanese 代表語の評価

- 代表語「互いに」は **相互(one another / each other)を表す副詞的表現**で、全 100 件に適合する。
- 実 Engine 出力(実測): 対格/属格/与格いずれも **「互いに」**(末尾「に」が格助詞ガードに該当し、
  格助詞は付与されず fallback。破損なし)。
- 「互いに」は gender/number/person/case を通じて **一定で安全**な代表語であり、H-3b の
  「安定した代表語」として妥当。**維持**する。

---

## 4. αὐτός・再帰代名詞との責務差分

| 観点 | αὐτός(G846) | 再帰 ἐμαυτοῦ/σεαυτοῦ | **ἀλλήλων(G240)** |
|---|---|---|---|
| 変化の駆動軸 | gender / number(morph 明示) | person(strong で一意) | **なし(全軸が不変/中立)** |
| gender | 語幹を決める(彼/彼女/それ) | 中立 | **中立**(互いに) |
| number | 語幹を決める(彼/彼ら) | 常に単数 | **常に複数・不変** |
| person | — | strong で一意(私自身/あなた自身) | **中立・非マーク** |
| case | 格助詞(の/に/を) | 格助詞(既存) | **副詞的「互いに」で格助詞非該当** |
| Morph の仕事 | 中核(誤り訂正) | person 精密化 | **なし** |

- αὐτós は形態(gender/number)で分岐、再帰は person(strong)で分岐。**ἀλλήλων はどの形態軸でも
  分岐しない**(常に複数・person/gender 中立・副詞的)。**Morph が扱う「形態による語幹変化」が
  そもそも存在しない**。

---

## 5. Morph 担当可能範囲の判定

判定基準: **形態情報だけで一意に語幹変化できるもの**。

| 形態軸 | 変化の有無 | Morph 可否 |
|---|---|---|
| gender | 無(互いに は gender 中立) | 不要 |
| number | 無(常に複数・不変) | 不要 |
| person | 無(中立・非マーク) | 不要 |
| case | 副詞的「互いに」で格助詞非該当(末尾「に」ガード)。案(互い+格助詞)は Data 代表語変更を要し、かつ case→役割は Syntax | 対象外 |

**結論: ἀλλήλων に Morph が担当できる語幹変化は存在しない。** 全形態軸が不変または中立であり、
「形態情報だけで一意に語幹変化できるもの」に該当しない。**Morph Rule 対象外**。

---

## 6. Semantic / Syntax 担当範囲

将来仮に精密化するとしても(本監査の対象外・実装しない):

| 領域 | 責務 | 内容 |
|---|---|---|
| case→文法役割 | **Syntax** | 対格(互いを=相互の目的語)・属格(互いの=相互の所有)の精密化。ただし Data 代表語を「互い」へ変える必要があり、かつ相互の役割は節構造依存 |
| 相互関係の同定 | **Semantic** | 誰と誰が相互作用するか(先行詞・文脈)。「互いに」は副詞的にこれを包含 |
| 強調 | Semantic | 「互いに〜し合う」等の強調 |

- いずれも **「互いに」が副詞的に基本的な相互性を表現できている**ため、精密化の緊急度は低い。

---

## 7. 実装価値の判定

- **実装価値: なし(Morph Rule 対象外)**。ἀλλήλων は全形態軸が不変/中立で、Morph が変化させる
  対象がない。代表語「互いに」は全 100 件で適合し破損もない。
- これは「未整備の layer に阻まれている」(指示詞=Syntax 依存)でも「文脈で不一意」(ἑαυτοῦ=Semantic)
  でもなく、**そもそも形態的変化が存在しない定数**であることによる対象外である。
- したがって **Morph Rule Engine v1 に ἀλλήλων を含めない**。

---

## 8. 実装 Gate 判定

| Gate | 状態 | 理由 |
|---|---|---|
| G-morph(形態で一意な語幹変化) | **不成立** | gender/number/person すべて不変・中立。変化軸なし |
| G-data(代表語妥当) | 充足 | 「互いに」は全件適合・維持 |
| G-value(実装価値) | **なし** | Morph の担当対象が存在しない |

### 判定

- **ἀλλήλων(G240)は Morph Rule 対象外**。**J-8c は実装に進まない**(実装不要の確定)。
- case→役割(Syntax)・相互関係(Semantic)は将来の別 layer の責務として保留するが、「互いに」で
  基本読解は成立しており、優先度は低い。
- **本監査により、Morph Rule Engine v1 の対象 lemma は確定**(αὐτós/τίス/ὅς/ὅστις/ὅσος/ἐμαυτοῦ/σεαυτοῦ の
  7 lemma)。指示詞(J-7a=Syntax 依存)・ἑαυτοῦ(J-8a=Semantic)・ἀλλήλων(本監査=変化なし)は
  いずれも Morph v1 の範囲外であることが確定した。
