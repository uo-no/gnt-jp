# 日替わり聖句 年間設計計画

**フェーズ**: D-1A  
**作成日**: 2026-07-15  
**種別**: 設計仕様書（実装変更なし・節の選定なし）  
**対象**: `DAILY_VERSES_NT` 配列の365節拡張設計  
**前提文書**: `todays-discovery-selection-spec.md`

---

## 1. 基本コンセプト

### 設計思想

日替わり聖句は単なる「人気聖句の循環」ではなく、**一年を通じて救済史（Heilsgeschichte）をたどる旅**として設計する。

読者は毎日1節に出会い、その節をキーワードで検索することで原語テキスト（NT/LXX）の実際の表現に触れる。Discovery の入口として機能するとともに、365日を通じて聖書の大きな物語の中での自分の位置を体感できる。

### 対象ユーザー

| ユーザー像 | 期待される体験 |
|-----------|-------------|
| 聖書を読み始めた人 | 毎日1節から入り、短い言葉を繰り返し味わう |
| 定期的に聖書を読む人 | テーマの流れを意識しながら、原語検索で深く掘り下げる |
| 研究・黙想を続ける人 | 年間構成を俯瞰し、救済史の流れの中で節を位置づける |

### 一年で得られる体験

```
1月  創造の光で始まる
  ↓
2-3月  神の愛と契約を深め、悔い改めへ
  ↓
4月  十字架と復活で中心に達する
  ↓
5-6月  新しいいのちと御霊による教会の誕生
  ↓
7-8月  弟子道・信仰・忍耐の夏
  ↓
9-10月  知恵と宣教の秋
  ↓
11月  感謝と終末的希望
  ↓
12月  待ち望みと降誕で年を閉じる
  ↓
再び1月  新しい被造物として次の年へ
```

## 4. 救済史との対応

年間を通じて NT が前提としている旧約の大きな物語を、NT の言葉を通して辿る。

```
創造（1月）
  ヨハネ 1:1 "初めに言があった"
  → 創世記 1 の再演。年の始まりと創造の始まりが重なる。

堕落と回復の必要（3月前半）
  ロマ 3:23 "すべての人は罪を犯した"
  → 四旬節の文脈。自己認識から悔い改めへ。

契約（2月・10月）
  ヘブル 8:10 "わたしは彼らと新しい契約を結ぶ"
  → 旧契約の成就としての新契約。愛と宣教の根拠。

出エジプトの型（4月）
  コリント第一 5:7 "キリストは私たちの過越の子羊"
  → 十字架は新しい出エジプト。過越から聖週間へ。

王国（9月・11月）
  マタイ 6:33 "まず神の国を求めなさい"
  ルカ 17:21 "神の国はあなたがたの中にある"
  → ダビデの王国から神の国の倫理と終末的実現へ。

預言の成就（12月）
  マタイ 1:22-23 "インマヌエル—神われらとともに"
  → アドベントは旧約預言の長い沈黙の終わりとして読む。

キリスト（4月・年間全体）
  全節のキリスト中心性。年間の軸。

教会の誕生（6月）
  使徒 2:1-4 「聖霊が降られた」
  → ペンテコステにより救済史の新段階が始まる。

新天新地（11月末・12月末）
  黙示録 21:4-5 "見よ、すべてを新しくする"
  → 年の終わりは終末の希望と次の始まりに向かう。
```

## 5. 教会暦との対応方針

### シーズン対応表

| 教会暦シーズン | 対応月（目安） | 年間設計との対応 | 実装注記 |
|-------------|-------------|---------------|---------|
| **Advent** | 12月 Week 1-4 | 第12月全体が待降節精神 | 変動あり（11月末〜） |
| **Christmas** | 12月 Week 4 | 受肉・光・讃美 | 12/25 固定節推奨 |
| **Epiphany** | 1月 Week 1-2 | 「光が来た」テーマ | 1/6 固定 |
| **Lent** | 3月（一部 2月末） | 第3月の全体がLent精神 | 変動あり（灰の水曜から） |
| **Holy Week** | 4月 Week 2-3 | 受難週への集中 | 変動あり |
| **Easter** | 4月 Week 4-5 | 復活のテーマ | 変動あり |
| **Pentecost** | 6月 Week 1 | 聖霊降臨の週 | Easter+50日 |
| **Ordinary Time** | 7〜11月（大部分） | 弟子道・宣教・成長 | 上記以外の全期間 |

### 変動シーズンの実装方針

教会暦の主要節（Lent・Easter・Pentecost）は復活祭（可動祝日）に依存するため、カレンダー月との対応は年によって最大4週ずれる。

**Phase D-1A（現段階）の方針：**

```
月ベース固定（近似）を採用する。
  3月  → Lent 精神
  4月  → Easter 精神
  6月1週 → Pentecost 精神
  12月 → Advent / Christmas 精神

正確な教会暦計算は Phase D-3 以降で実装する。
実装方法は todays-discovery-selection-spec.md §4 の
getCurrentSeason() 拡張と同期する。
```

**Phase D-3 以降の方針（参考）：**

Computus アルゴリズムで Easter Sunday を算出し、  
Lent（Easter-46日）、Pentecost（Easter+49日）、Advent（11月30日以前の第4日曜）を動的に判定する。

## 6. 日本の季節との対応

日本の四季は教会暦と重なりながらも独自のリズムを持つ。  
聖句の選定とテーマ配置において、日本の文化的感受性と自然の流れを活かす。

### 季節×テーマ対応

| 季節 | 月 | 自然のテーマ | 聖句テーマとの共鳴 |
|-----|---|------------|----------------|
| **冬** | 12〜2月 | 寒さ・静寂・光を待つ | 待ち望み・光・愛の温かさ |
| **春** | 3〜5月 | 桜・卒業・入学・新生 | 悔い改め・復活・新しいいのち |
| **初夏** | 6〜7月 | 梅雨・成長・活動 | 御霊・弟子道・実を結ぶ |
| **盛夏** | 8月 | 暑さ・試練・盆 | 忍耐・信仰・死と復活（お盆の死生観と） |
| **秋** | 9〜11月 | 実り・収穫・学び・感謝 | 知恵・宣教・感謝・希望 |

### 季節的配慮のある節の例

| 季節 | 配慮すべき文化的文脈 | 推奨するテーマ |
|-----|-------------------|-------------|
| 1月元旦 | 新年の誓い・初詣 | 新しい創造・「見よ、すべてを新しくする」 |
| 3月卒業期 | 別れと旅立ち | 「恐れるな、わたしはあなたとともにいる」 |
| 4月入学期 | 始まりの不安 | 「わたしはあなたがたを安心させる」 |
| 8月お盆 | 先祖・死者への思い | 復活の希望・「死は飲み込まれた」 |
| 11月感謝 | 収穫感謝（勤労感謝） | 「すべてのことに感謝せよ」 |
| 12月クリスマス | 国民的行事としての知名度 | 受肉・「インマヌエル」 |

## 7. 品質基準

365節の選定にあたり、以下の基準をすべて満たすこと。

### 必須条件（Mandatory）

| # | 基準 | 検証方法 |
|---|-----|---------|
| M-1 | **NT優先**: 全節が `corpus: 'NT'`。LXX対応後は `corpus: 'LXX'` を追加可。 | corpus フィールド確認 |
| M-2 | **本文との整合**: `keywords` の各語が表示行（`lines`）に実際に存在する | 目視確認 |
| M-3 | **検索可能**: `keywords` の全語が `_exampleSearch()` で 1件以上ヒット | 自動検証スクリプト実行 |
| M-4 | **重複なし**: 365節において同一 `ref` の重複がない | 自動検証スクリプト実行 |
| M-5 | **表示長**: `lines` が 2〜4行に収まる（モバイル画面で快適に読める長さ） | 目視確認 |
| M-6 | **keywords 件数**: 1〜5語の範囲（最低1語。5語を超えない） | 自動検証 |

### 推奨条件（Preferred）

| # | 基準 | 考慮のポイント |
|---|-----|-------------|
| P-1 | **書の分布**: NT全体から幅広く選ぶ（下表参照） | 特定の書への偏りを避ける |
| P-2 | **月テーマとの整合**: 選んだ節がその月の大テーマと明確につながる | 週テーマとの一致を優先 |
| P-3 | **keywords の多様性**: 年間を通じて「神・愛・心」に偏りすぎない | 30節パイロットの反省 |
| P-4 | **易しさと深さのバランス**: 親しみやすい節と、原語検索で深みが出る節を混在 | 例：罪→ἁμαρτία の用法探索 |
| P-5 | **教派的普遍性**: 特定の教派解釈に強く依存しない節を優先 | 洗礼・聖餐論争節は慎重に |
| P-6 | **日本語の美しさ**: 新改訳2017を基準とし、日本語として耳に残る表現 | 音読したときの自然さ |
| P-7 | **救済史との接続**: 年間の大きな流れの中での節の位置が明確 | Section 4 の対応表を参照 |

| 福音書 | 約40% | ~146節 | マタイ・マルコ・ルカ・ヨハネ（均等） |
| パウロ書簡 | 約35% | ~128節 | ロマ・コリント・ガラテヤ・エペソ・ピリピ・コロサイ・テサロニケ・テモテ・テトス |
| 公同書簡・ヘブル | 約15% | ~55節 | ヘブル・ヤコブ・ペテロ・ヨハネ書簡・ユダ |
| 使徒の働き | 約5% | ~18節 | 使徒 |
| 黙示録 | 約5% | ~18節 | 黙示録 |

### keywords 検証フロー

```
① 候補節を決定
② DAILY_VERSE の形式でデータを記述
③ 自動検証スクリプト（verify-keywords.mjs 相当）を実行
   → 全 keywords が 0件でないことを確認
④ 0件が出た場合:
   a. keywords を変更する（本文に現れる別の語を選ぶ）
   b. または 0件になる語を除外して別語を追加する
   c. どうしても適切な keywords が見つからない節は別の節に差し替える
⑤ 全節合格後に実装へ
```
