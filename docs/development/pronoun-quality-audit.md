## 6. 結論(責務分類のみ・修正は行わない)

1. **純データ誤り 1 件を検出**: σύ(G5213)= 「彼」は 2 人称複数「あなたがた」を
   3 人称男性単数で表記する固定値誤り(607 件・不適合 100%)。σύ 他形はすべて「あなた」で、
   lemma 内一貫性も欠く。**μόδιος(Stage H-2)と同型の Data 責務**。
2. **最大規模の Type C は αὐτός(G846)**: count 5,067・不適合 1,888 件(37.3%)。
   gender/number を単一固定訳「彼」では表現できず、Morph Engine の選択が必要。
3. **人称代名詞の単数(ἐγώ 私・σύ あなた)は Type A で安定**。ただし**複数は「私たち/あなたがた」
   の number が非反映**で、これは Data ではなく Morph Engine の領域。
4. **指示詞(οὗτος この・ἐκεῖνος あの)は Type B〜C**。連体用法は代表語で可、代名詞用法と
   discourse 距離は Engine/Semantic が担う。
5. **bible_data.japanese の責務は「安定代表語の保持」**(H-3b ポリシー)。Type C でも japanese
   削除ではなく、変化を Engine/Semantic に委ねる責務分離が妥当。**例外は σύ G5213 のみ**で、
   これは代表語自体が誤っているため Data 修正対象(本監査は分類のみ・修正は次段階)。
