## lemma 別 Morph 責務定義

各 lemma について ① Morph だけで決定できる情報、② Semantic へ委譲する情報 を定義する。

### τίς(疑問詞)

- ① **Morph 決定**: gender で人/事物を分岐。**masculine/feminine → 誰(人)**、**neuter → 何(事物)**。
  number で 誰/誰たち(まれ)。gender は token が保持する形態値であり、推論不要で一意。
- ② **Semantic 委譲**: neuter τί の**語用**(なぜ=why 99 / どれ=which 35)。同じ neuter でも
  「何を」か「なぜ」かは格・文脈の語用で決まり、形態だけでは一意化できない。

### ὅς(関係詞)

- ① **Morph 決定**: gender で referent の人/物を分岐。**M/F → 者(人)**、**N → もの/こと(事物)**。
  固定「〜する者」は人限定で中性(N652)に不適合 → gender で 者/もの を選ぶのが Morph 責務。
  number(単複)も形態で反映。
- ② **Semantic / Syntax 委譲**: **関係節の構文機能**(先行詞との連結・substantive 化)は Syntax。
  不定関係(whoever)や自由関係の語義は Semantic。Morph は語幹(者/もの)の性数選択に閉じる。

### ὅστις(不定関係詞)

- ① **Morph 決定**: gender で 者(M/F 189)/もの(N5)。number 反映。
- ② **Semantic / Syntax 委譲**: **不定関係「だれでも/whoever」**(既に ja に だれでも 8 件)は語義=Semantic。
  関係節構文は Syntax。ὅς と同型で、Morph は性数の語幹選択のみ。

### ὅσος(数量関係詞)

- ① **Morph 決定**: gender/number で人/事物・単複を分岐(**M → 者、N → もの/こと**、多くは N/plural 81=
  「〜するものすべて」)。
- ② **Semantic 委譲**: **数量・程度の語義**(whatever / how much / as much as / all)。「どれだけの量か」は
  形態では決まらず語義選択=Semantic。関係節構文は Syntax。

### ἐκεῖνος(遠称指示詞)

- ① **Morph 決定**: 代名詞用法での性数語幹。**M/F → あの人(々)**、**N → あれ / あれら**、number で
  単複。連体用法「あの」は代表語のまま。
- ② **Semantic 委譲**: **談話距離**(この/その/あの の選択)。ἐκεῖνος は遠称固定だが、日本語の
  この/その/あの は談話上の距離・既出性で選ばれるため Semantic。指示対象の同定も Semantic。

### οὗτος(近称指示詞)

- ① **Morph 決定**: number で **この(単)/これらの(複)**(既に ja に これらの 316 件)。代名詞用法の
  性数語幹 **N → これ / これら**、**M/F → この人(々)**。
- ② **Semantic 委譲**: **談話上の指示対象**(何を指す「これ」か)、近称の含意、「すなわち(=)」用法
  (ja に 19 件)は語義=Semantic。

### τοιοῦτος(性状指示詞)

- ① **Morph 決定**: gender/number で **M/F → このような人(々)**、**N → このようなもの**。連体
  「このような」は代表語のまま。
- ② **Semantic 委譲**: **質・種類の含意**(such / of a kind / 「そのような性質の」)。程度・比較の
  ニュアンスは語義=Semantic。

## ③ Morph Rule Table(Data / Morph / Semantic)

| lemma | Data 代表語 | Morph が担当(形態で決定) | Semantic が担当(語義・談話・構文) |
|---|---|---|---|
| **τίς** | 誰 | gender: M/F→誰・N→何 / number | なぜ(why)・どれ(which)の語用 |
| **ὅς** | 〜する者 | gender: M/F→者・N→もの / number | 関係節構文(Syntax)・不定/自由関係 |
| **ὅστις** | 〜する者 | gender: M/F→者・N→もの / number | 不定関係(だれでも)・関係節構文(Syntax) |
| **ὅσος** | 〜するだけ | gender/number: 者/もの・単複 | 数量・程度の語義・関係節構文(Syntax) |
| **ἐκεῖνος** | あの | 代名詞語幹 M/F→あの人・N→あれ / number | 談話距離(この/その/あの)・指示対象同定 |
| **οὗτος** | この | number→この/これらの・代名詞語幹 N→これ(ら)・M/F→この人 | 指示対象・近称含意・すなわち用法 |
| **τοιοῦτος** | このような | gender/number: このような人/もの・単複 | 質・種類の含意(such) |

- **Data は各 lemma 1 つの代表語を維持**(H-3b)。削除も多義化もしない。
- **Morph は gender/number(および case の既存助詞付与)で語幹を分岐**。入力は形態値のみ。
- **関係詞(ὅς / ὅστις / ὅσος)は Morph(性数)+ Syntax(関係節)の協働**が必要で、Morph 単独では
  完結しない。Rule Table 上は Morph 担当分(者/もの)と Syntax/Semantic 担当分を分離して登録する。

## ④ Rule 適用条件

I-2(morph-rule-engine-design.md)の帰属基準を本ファミリーに適用する。

### Morph Rule を適用してよい条件(すべて満たす)

1. **形態情報(gender / number / case / person)だけで出力語幹が一意に決定できる**。
2. **意味推論・文脈参照・先行詞解決・談話判断を要しない**。
3. **決定的**(同一形態 → 常に同一語幹)。

本ファミリーで該当するのは各 lemma の **gender による人/事物の分岐(誰/何・者/もの・あの人/あれ)**
と **number による単複**である。これらは token の形態値から一意で、推論を要さない。

### Semantic へ委譲する条件(いずれか該当)

1. 同一形態でも**語義**で出力が変わる(τί の なぜ/何、ὅσος の数量語義)。
2. **談話**(この/その/あの の距離、指示対象)の解釈を要する(ἐκεῖνος / οὗτος)。
3. **関係節の構文機能**(先行詞連結・substantive 化)を要する(ὅς / ὅστις / ὅσος → Syntax)。
4. **不定・自由関係・強調・idiom** など、形態に還元できない判断を要する。

> 一語要約: **形態で一意=Morph / 語義・談話・関係節構文=Semantic(または Syntax)**。

## ⑤ 一般化

- 本 7 lemma はいずれも **「Data 代表語 → Morph が gender/number で語幹分岐 → Semantic/Syntax が
  語義・談話・構文を分離」** という I-2 の共通枠組みに収まる。特別扱いは不要。
- 新しい lemma を追加する場合も、次を確認するだけで同じ枠組みに登録できる:
  1. Data 代表語は 1 つか(H-3b)。
  2. gender/number で一意に決まる語幹分岐はどれか(→ Morph、④の 3 条件)。
  3. 形態で決まらない語義・談話・構文はどれか(→ Semantic / Syntax)。
- **Rule Table(③)の 1 行を追加するだけで拡張でき、Engine の構造・責務境界・パイプラインは
  変更しない**(I-2 の拡張方針)。関係詞のように Syntax 協働が要る lemma も、Morph 担当分と
  Syntax 担当分を分けて登録する同一手順で扱える。
- これにより H-3b で Type C(固定禁止)と分類された疑問詞・関係詞・指示詞群は、**Data の削除では
  なく責務の再配置**で一貫して扱える。

## 責務確定(要約)

| lemma | Data | Morph | Semantic / Syntax |
|---|---|---|---|
| τίς | 誰(代表語) | 性→誰/何・数 | 語用(なぜ/どれ) |
| ὅς | 〜する者 | 性→者/もの・数 | 関係節構文(Syntax) |
| ὅστις | 〜する者 | 性→者/もの・数 | 不定関係・関係節構文 |
| ὅσος | 〜するだけ | 性数→者/もの | 数量語義・関係節構文 |
| ἐκεῖνος | あの | 代名詞語幹(あの人/あれ)・数 | 談話距離・指示対象 |
| οὗτος | この | この/これらの・代名詞語幹 | 指示対象・すなわち |
| τοιοῦτος | このような | 性数→このような人/もの | 質・種類の含意 |

**確定**: 7 lemma すべてで **Data は代表語を維持し、Morph は gender/number による語幹分岐を担い、
形態で決まらない語義・談話・関係節構文は Semantic(関係詞は Syntax 協働)へ委譲する**。本設計は
責務定義のみで、実装・データ修正は行わない。
